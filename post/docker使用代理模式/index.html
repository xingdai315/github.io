docker使用代理模式

基于systemd的override方式

systemctl edit docker

填写代理信息

[Service]
Environment="HTTP_PROXY=http://192.168.31.199:8118/" "HTTPS_PROXY=http://192.168.31.199:8118/"
"NO_PROXY=localhost,127.0.0.1,registry.docker-cn.com"

读取配置&重启

systemctl daemon-reload && systemctl restart docker
